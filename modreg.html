CS1101S Cheatsheets

Welcome to Discount ModReg!

This is an experimental site to teach freshmen how to use our Module Registration (ModReg) System.

Bi-annually, all SOC students scramble to ModReg to select our modules and tutorials/labs
for the coming semester. There are multiple rounds of "bidding" for modules so it tends to get
pretty competitive, especially for popular, in-demand modules.

Hope this site helps you familiarise with ModReg before it's your turn very soon!


"HAPPY BIDDING!"
~ SOC FOP OComm



DISCLAIMER
This website, by no means, is an alternative to, or seeks to copy, myEduRec. Please use the official
Module Registration system and the facilities provided by NUS for all student-related activities.

Please note that there will be no classes on public holidays.

The module instructor(s) will advise on makeup classes, if any.
